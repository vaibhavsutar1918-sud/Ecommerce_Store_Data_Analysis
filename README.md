# Ecommerce_Store_Data_Analysis
Built a Power BI dashboard to analyze e-commerce sales data, track key business metrics, and generate actionable insights for decision-making
This project focuses on analyzing e-commerce store sales data using Power BI. The dashboard provides valuable insights into sales performance, customer behavior, product categories, and regional trends. The goal is to help businesses make data-driven decisions by identifying key patterns and opportunities for growth.

**🎯 Objectives**
Analyze overall sales performance.
Track profit and revenue trends.
Identify top-selling products and categories.
Understand customer purchasing behavior.
Compare sales across different regions.
Monitor key business KPIs.

**🛠 Tools & Technologies**

Power BI
Power Query
DAX (Data Analysis Expressions)
Microsoft Excel / CSV

**📊 Dashboard Features**

Total Sales Analysis
Profit Analysis
Category-wise Sales Performance
Regional Sales Distribution
Customer Insights
Monthly and Yearly Sales Trends
Interactive Filters and Slicers

**📈 Key Insights**

Identified top-performing product categories.
Analyzed sales and profit trends over time.
Determined high-revenue regions and customer segments.
Highlighted areas for business improvement and growth.

**📂 Dataset**

The dataset contains:

Order Details
Customer Information
Product Categories
Sales and Profit Data
Regional Information
**🚀 Project Outcome**

Developed an interactive Power BI dashboard that transforms raw e-commerce data into meaningful business insights, enabling stakeholders to make informed decisions and improve overall business performance.
